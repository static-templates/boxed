More Details

This template is built with Static. It uses TailwindCSS and a little bit of AlpineJS. This is the perfect template for your next blog or personal website. Be sure to learn more by visiting the template repo at https://github.com/static-templates/boxed.

Contributing

If you feel like this template could use some improvements, be sure to submit a PR and leave a detailed explanation of your contribution and we would love to have it integrated into this template.
